# DynamicWebassemblyHPC
This Project provides a Dynamic High Performance Computing Network

## Frontend (Next.Js)
- Port: 4000
- Run Dev: `npm run dev`
- Run Dep: `npm run start`

## Backend (Nest.Js)
- Port: 3000
- Run Dev: `npm run start:dev`
- Run Dep: `npm start`
- Build: `npm build`
- Prod: `npm run start:prod`

## Database (Postgres)
- Port: 5432
- Run: `docker compose up`


## Notes:
- Device IP: `hostname -I`
